Installation

  1. Install the uget-chrome-wrapper

    
      yaourt -S uget-chrome-wrapper
    
    
      packer -S uget-chrome-wrapper
    
    
          sudo add-apt-repository ppa:slgobinath/uget-chrome-wrapper
          sudo apt update
          sudo apt install uget-chrome-wrapper
      
    
            wget https://raw.githubusercontent.com/slgobinath/uget-chrome-wrapper/master/build/linux/install_uget_chrome_wrapper.sh
            sudo sh install_uget_chrome_wrapper.sh
    1. Install Python 3.4 or latest
    2. Download and extract uGet to C:\uget folder
    3. Add the Python installation directory and C:\uget\bin folder to the environment variable PATH
    4. Download and install the latest uget-chrome-wrapper_x.x.x.x.exe
    5. Restart the computer

  2. Install the uGet Integration extension in your browser

    • Google Chrome, Chromium & Vivaldi users: Chrome Extension
    • Opera users: Opera Extension
    • Mozilla Firefox 56+ users: Firefox Addon
